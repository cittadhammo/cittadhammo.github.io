Use the download links on each chart page to get the best file for printing.

## Download formats

Each chart page includes a list of available files. Click the appropriate format to download it.

- **PNG** Raster downloads for quick viewing and smaller prints. Some PNG files can be very large.
- **PDF** Print-ready output where available.
- **SVG** Vector downloads, useful for browser printing and scaling.
- **ONLINE** Interactive previews on supported charts.

Tip: Hover your mouse over a file format to see its file size.

## Minimum resolution for 300 DPI

Use the table below as a guide when printing from a raster file.

| Paper Size | Millimetres (mm) | Inches | Pixels at 300 DPI |
|------------|------------------|--------|-------------------|
| A0         | 841 x 1189       | 33 x 47 | 9933 x 14043     |
| A1         | 594 x 841        | 23 x 33 | 7016 x 9933      |
| A2         | 420 x 594        | 16.5 x 23 | 4961 x 7016   |
| A3         | 297 x 420        | 11.5 x 16.5 | 3508 x 4961  |

## Printing at home

[The Rasterbator](https://rasterbator.net/) can split a large image into pages for home printing.

## Printing online

You can use an online printing service to get a poster printed and delivered. Many also sell frames.

- [Doxdirect](https://www.doxdirect.com/) for poster printing
- [Mixam](https://mixam.co.uk/) for poster printing
- [A0 Size Calculator](https://www.a0-size.com/a0-size-in-pixels/?size=a0&unit=px&ppi=300) for checking pixel dimensions
