## Download formats


* **PDF** (Recommended) Print-ready output. This vector format ensures the sharpest possible text and lines at large scales.
* **SVG** Vector format, recommended for custom scaling or printing directly from a web browser.
* **PNG** Raster format for quick viewing and smaller prints. Please note that some high-resolution PNG files can be very large.
* **ONLINE** Interactive previews for supported charts.

> **Tip:** Hover your mouse over a file format to check its file size.


## Minimum resolution for 300 DPI

Use the table below as a guide when printing from a raster (PNG) file. Vector formats like PDF and SVG can be scaled indefinitely without loss of quality.

| Paper Size | Millimetres (mm) | Inches | Pixels at 300 DPI |
| :--- | :--- | :--- | :--- |
| **A0** | 841 x 1189 | 33 x 47 | 9933 x 14043 |
| **A1** | 594 x 841 | 23 x 33 | 7016 x 9933 |
| **A2** | 420 x 594 | 16.5 x 23 | 4961 x 7016 |
| **A3** | 297 x 420 | 11.5 x 16.5 | 3508 x 4961 |
