Each chart now has its own item page on the website.

Open the chart you want, then use the download links in the item details. Most chart pages provide:

- `PNG` for raster downloads
- `PDF` for print-ready output where available
- `SVG` for vector downloads and browser printing
- `ONLINE` for interactive map-style previews on supported charts

If a chart has multiple variants, choose the version you want on the item page first, then download that file.

## Recommended print workflow

1. Open the chart item page.
2. Prefer `PDF` if it is available.
3. If there is no PDF, use `SVG`.
4. Open the file in your browser or PDF viewer.
5. Print at the paper size you want, such as A1 or A0.

## Minimum Resolution For 300 DPI

Use the table below as a guide when printing from a raster file.

| Paper Size | Millimetres (mm) | Inches | Pixels at 300 DPI |
|------------|------------------|--------|-------------------|
| A0         | 841 x 1189       | 33 x 47 | 9933 x 14043     |
| A1         | 594 x 841        | 23 x 33 | 7016 x 9933      |
| A2         | 420 x 594        | 16.5 x 23 | 4961 x 7016   |
| A3         | 297 x 420        | 11.5 x 16.5 | 3508 x 4961  |
